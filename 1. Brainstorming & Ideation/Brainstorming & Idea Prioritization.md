# Project Phase 1: Brainstorming & Idea Prioritization

### 1. High Priority (Must Implement First)
* **Data Prep**: Merge raw customer data files, clean up missing structural records, and apply label encoding to clean non-numeric fields.
* **Core ML Classification Models**: Train foundational Logistic Regression and Random Forest architectures to evaluate a baseline accuracy.

### 2. Medium Priority (Should Implement)
* **Optimization**: Balance dataset class distributions and execute Hyperparameter tuning via GridSearchCV.
* **Metric Plots**: Generate Confusion Matrices and ROC-AUC curve visualizations for model transparency.

### 3. Low Priority (Future Scope)
* **Web App Deployment**: Wrap the model pipeline into a functional web application via Streamlit or a FastAPI backend.
