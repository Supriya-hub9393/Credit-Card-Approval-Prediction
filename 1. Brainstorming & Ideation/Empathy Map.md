# Project Phase 1: User Empathy Map
**Target Persona:** Bank Credit Risk Evaluation Officer

 Quadrant  Observation Details 

SAYS: "We have to sort through thousands of applicant profiles daily while keeping default rates low." 
| **THINKS** | "Is this applicant's debt profile safe? Verification takes too long; I wish we could instantly view data patterns." |
| **DOES** | Manually cross-checks income reports, analyzes debt ratios, verifies job tenure logs, and logs inputs in spreadsheets. |
| **FEELS** | Stressed by application backlog stress, and constantly worried about approving risky profiles that could lapse into bad debt. |
