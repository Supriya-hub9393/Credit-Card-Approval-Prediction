# Credit Card Approval Prediction System

An intelligent web-based machine learning application to predict whether a credit card application will be approved or rejected using localized parameters.

## How to Activate and Run the Application Locally

Follow these precise step-by-step terminal commands to turn on the project dashboard:

### 1. Install Required Packages
Run this command from the project root folder to install all necessary Python data science and web libraries:
```bash
pip install Flask numpy scikit-learn xgboost pandas
```

### 2. Move Into the Phase 5 App Directory & Launch the Server Engine
Navigate into the active application folder & Execute the main server controller script to launch the Flask framework local loop:
```bash
cd "5. Project Development Phase" && python3 app.py
```

### 3. Open in Your Browser
Once the terminal reads `Running on http://127.0.0.1:5000`, open your web browser and navigate to:
```text
http://127.0.0.1:5000
```
